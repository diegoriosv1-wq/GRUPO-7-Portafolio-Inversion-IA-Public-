Descripción del proyecto

Calculadora de Portafolio de Inversión basada en Inteligencia Artificial y Computación en la Nube

Este proyecto consiste en el desarrollo de una aplicación web sencilla creada con HTML, CSS y JavaScript, cuyo objetivo es ayudar al usuario a distribuir su capital de inversión entre empresas líderes del sector tecnológico. La propuesta se basa en un portafolio diversificado conformado por Microsoft (35 %), Amazon (25 %), Nvidia (25 %) y el ETF Invesco QQQ (15 %), buscando maximizar la rentabilidad y reducir el riesgo mediante una adecuada diversificación.

La aplicación permite al usuario ingresar un monto de inversión y, de forma automática, calcula cuánto debe destinar a cada activo según la distribución establecida. Además, muestra una estimación de la rentabilidad anual, el capital final esperado y una recomendación de inversión de acuerdo con el monto ingresado.

Este proyecto fue desarrollado como parte del trabajo final del curso de Microeconomía, integrando conceptos financieros con herramientas de desarrollo web para presentar una solución práctica, interactiva y fácil de utilizar.
